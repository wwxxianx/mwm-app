Read Me (Setup Instruction)

To run this project locally:

  1. Provide a valid SQL database connection string in the appsettings.json file located in the root level of the server folder (as shown in the figure below).

  2. Open CLI and navigate to the root level of the server folder.
  3. Run command “dotnet run”.
  4. The application will be hosted on https://localhost:5173

Note:

  • Testing data is needed to run this project without any error. This includes
    1. One admin account in order to login to admin dashboard
    2. Books data to be displayed
